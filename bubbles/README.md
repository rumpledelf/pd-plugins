# Bubbles Plugin

An interactive canvas-based plugin where bubbles rise from the bottom-left to the top-right, swaying gently as they float.

## Features

- **Drifting Physics**: Bubbles float upward and to the right with custom sinusoidal wobble simulation to mimic real air movement.
- **Bubble Specular Highlights**: Beautiful 3D rendered sphere appearance with multi-layered specular gradients using core host palette themes (Magenta, Purple, Blue).
- **Pop Interaction**: Clicking bubbles pops them with particle explosions and shockwave rings.
- **Pop Speed Scaling**: Popping bubbles increases speed multiplier dynamically by 0.05x.
- **Controls**: Native-styled control bar for Start, Stop, and Restart actions.
