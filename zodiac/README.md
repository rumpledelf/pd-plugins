# Zodiac

This is a small single-purpose page module.

The user enters their birthday with a date picker and the module tells them their zodiac sign. The result links to the corresponding Photographic Dictionary page for that sign, for example `/g/gemini`.

This family is intentionally simple and bounded:

- one variant
- one page target
- one injected instance
- static HTML snippet
- JavaScript-owned behaviour
- no module CSS for now

It is the likely first canonical example for proving the page-module contract and the host-side bridge.
