# Piano

This is a tiny interactive one-octave piano module.

The user clicks the keys and hears the corresponding note. It is intentionally simple:

- one octave
- click-only interaction
- no score, quiz, or music composition features
- small static HTML snippet
- module-owned CSS for the keyboard layout
- JavaScript-owned sound generation using the browser audio API

This family exists to be playful rather than instructional. It is one of the clearest examples of the wider goal of this workspace: cute little interactive things that can sit on dictionary pages.
