# Bounce Plugin

This plugin features a physics-based interactive bouncing ball. The ball is colored magenta.

## Features

- Physics-based bouncing simulation with gravity and elasticity.
- Squash-and-stretch animation effects on bounce impact and during flight.
- Realistic dynamic shadow that scales and fades depending on the ball's altitude.
- Interactive controls to pause, resume, or reset the animation.
- Interactive touch and click support to place/drag/toss/flick the ball around the stage.
